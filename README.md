# codeLingo

🦉 CodeLingo - Aprenda Programação Jogando
O CodeLingo é uma plataforma de aprendizado gamificada inspirada no Duolingo, projetada para tornar o estudo de programação viciante, prático e divertido. O objetivo é quebrar conceitos complexos em lições rápidas e interativas.
🎯 O Desafio
Aprender programação pode ser intimidador. Este projeto resolve isso aplicando psicologia de jogos (gamificação) para manter o engajamento do estudante através de recompensas visuais e feedback imediato.
✨ Funcionalidades Principais

    Trilhas de Aprendizado: Lições sequenciais que cobrem desde lógica básica até frameworks modernos.
    Sistema de Estalecas (Streaks): Contador de ofensiva diária para incentivar a constância.
    Feedback Interativo: Validação de respostas em tempo real com efeitos sonoros e visuais.
    Interface Adaptativa: Design focado na experiência do usuário (UX), simulando a fluidez de um aplicativo mobile no navegador.

🛠️ Tecnologias Utilizadas

    React.js: Para a construção de uma interface componentizada e veloz.
    Styled Components: Estilização avançada e temas dinâmicos.
    Framer Motion: Responsável pelas animações suaves de transição entre lições.
    Lucide React: Ícones modernos e minimalistas.
